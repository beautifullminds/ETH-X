# 🌍 Overview

ETH-X is an open-source Web3 ecosystem focused on delivering a modern decentralized experience for users, developers, and businesses.

Our mission is to simplify blockchain technology while providing secure, scalable, and beautiful decentralized applications powered by Ethereum.

The project combines modern frontend technologies with blockchain infrastructure to create a seamless Web3 experience.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/beautifulminds/ETH-X.git

cd ETH-X
```

---

## Install Dependencies

```bash
npm install
```

or

```bash
pnpm install
```

---

## Start Development

```bash
npm run dev
```

or

```bash
pnpm dev
```

---

## Build Production

```bash
npm run build
```
